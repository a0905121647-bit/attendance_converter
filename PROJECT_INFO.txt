【計算規則】
================================================================================

起算時間：
  - 預設：08:00（所有員工）
  - 特殊：陳品璇（考勤號碼 101）11:00

中間休息推估：
  - 打卡間隔 ≥ 30 分鐘 且 ≤ 120 分鐘
  - 前打卡時間在 10:30-14:30
  - 推估 < 60 分鐘計為 60 分鐘

工時計算：
  - 實際工時 = (下班時間 - 上班時間) - 休息分鐘
  - 加班時數 = 超過 8 小時的部分

【快速開始】
================================================================================

方式 1：Streamlit Cloud（推薦，最簡單）
  1. 訪問 https://streamlit.io/cloud
  2. 使用 GitHub 帳戶登入
  3. 點擊「New app」
  4. 選擇本倉庫和 app.py
  5. 點擊「Deploy」
  預計時間：1-2 分鐘

方式 2：本地運行
  1. python3 -m venv venv
  2. source venv/bin/activate
  3. pip install -r requirements.txt
  4. streamlit run app.py
  5. 打開 http://localhost:8501

方式 3：Render/Railway/Heroku
  詳見 DEPLOYMENT.md

【使用流程】
================================================================================

1. 準備 CSV 檔案
   - 欄位：姓名、考勤號碼、日期時間、簽到/退

2. 上傳檔案
   - 在「上傳與處理」頁籤選擇 CSV
   - 點擊「開始處理」

3. 查看結果
   - 在「預覽結果」頁籤查看計算結果
   - 可按員工/日期篩選

4. 下載匯出
   - 在「下載匯出」頁籤下載 Excel
   - 匯入文中系統
