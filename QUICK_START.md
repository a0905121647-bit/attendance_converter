# 快速開始指南 - 5 分鐘上手

## 🚀 30 秒快速部署

### 選項 1：Streamlit Cloud（推薦，最簡單）

1. 訪問 https://streamlit.io/cloud
2. 使用 GitHub 帳戶登入
3. 點擊「New app」
4. 選擇本倉庫和 `app.py`
5. 點擊「Deploy」
6. **完成！** 應用將在 1-2 分鐘內上線

### 選項 2：本地運行

```bash
# 1. 克隆或下載專案
cd attendance_converter

# 2. 建立虛擬環境
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# 或
venv\Scripts\activate  # Windows

# 3. 安裝依賴
pip install -r requirements.txt

# 4. 運行應用
streamlit run app.py

# 5. 打開瀏覽器訪問 http://localhost:8501
```

## 📝 3 步使用系統

### 第 1 步：準備 CSV 檔案

建立一個 CSV 檔案，包含以下欄位：

```csv
姓名,考勤號碼,日期時間,簽到/退
王小明,001,2024-01-15 08:30,簽到
王小明,001,2024-01-15 12:00,簽退
王小明,001,2024-01-15 13:00,簽到
王小明,001,2024-01-15 17:30,簽退
```

**下載範例**：使用 `sample_attendance.csv`

### 第 2 步：上傳並處理

1. 打開系統 URL
2. 在「上傳與處理」頁籤選擇 CSV 檔案
3. 點擊「開始處理」
4. 等待完成（通常 < 1 秒）

### 第 3 步：下載結果

1. 進入「下載匯出」頁籤
2. 點擊「下載 Excel 檔案」
3. 將 Excel 匯入文中系統

## ⚙️ 關鍵設定（可選）

### 修改預設起算時間

側邊欄 → 系統設定 → 預設起算時間 → 修改小時/分鐘

### 為員工設定特殊班別

側邊欄 → 系統設定 → 新增員工特殊設定

範例：
- 考勤號碼：101
- 小時：11
- 分鐘：00
- 點擊「新增員工設定」

## 📊 計算規則速查

| 項目 | 說明 |
|------|------|
| **起算時間** | 預設 08:00，陳品璇（101）為 11:00 |
| **下班時間** | 當日最後一筆打卡 |
| **中間休息** | 打卡間隔 30-120 分鐘 + 前打卡在 10:30-14:30 |
| **最小休息** | 推估 < 60 分鐘計為 60 分鐘 |
| **實際工時** | (下班 - 上班) - 休息 |
| **加班時數** | 超過 8 小時的部分 |

## 🎯 常見操作

### 查看計算結果

1. 上傳後自動進入「預覽結果」
2. 可按員工或日期篩選
3. 查看詳細記錄和統計

### 修改計算參數

1. 側邊欄修改設定
2. 重新上傳檔案
3. 系統自動使用新參數

### 一次上傳多個檔案

1. 點擊「選擇一個或多個 CSV 檔案」
2. 按 Ctrl（或 Cmd）選擇多個
3. 點擊「開始處理」

## ❓ 常見問題速答

**Q：上傳失敗怎麼辦？**
A：檢查 CSV 是否包含必要欄位（姓名、考勤號碼、日期時間、簽到/退）

**Q：計算結果不對？**
A：檢查員工起算時間設定是否正確

**Q：如何修改起算時間？**
A：側邊欄 → 系統設定 → 預設起算時間

**Q：支援哪些日期格式？**
A：`2024-01-15 08:30`、`2024/01/15 08:30`、`01/15 08:30` 等多種

**Q：可以一次上傳多個檔案嗎？**
A：可以，系統會自動合併
